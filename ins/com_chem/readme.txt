Инструкция по установке и эксплуатации

1. Проинсталируйте приложение стандартными средствами Joomla
2. Перейдите в меню Components/Chem/Manage Molecules/Parameters и сохраните параметры

Просмотр информации о молекуле на сайте:
http://sitename/index.php?option=com_chem&id=0110420239
где id это каталожный номер

На фронтэнде доступна информация о стоимости вещества,
отображается молекула, есть возмодность загрузить SDF и MSDS файлы.

В админке доступна следующая функциональность:
1. просморт записи о молекуле
2. удаление записи
3. сортировка отображения записей
4. Экспорт записей в формат CVS по списку каталожных номеров
5. Импорт из SDF файла
6. Удаление всех записей из базы или удаление по спику каталожных номеров
7. Логирование операций импорта, удаления и экспорта
8. просмотри и удаление лог файлов

Размер импортируемого файла зависит от настроек сервера.
Если при операции импорта происходят ошибки то рекомендуется уменьшить объем импортируемого файла.
